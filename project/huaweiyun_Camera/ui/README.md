# 图像分块与整合工具

一个基于 PySide6 和 OpenCV 的图像处理工具，支持图像分块、整合、质量评估和可视化对比功能。

![PySide6](https://img.shields.io/badge/PySide6-1.0+-blue.svg)
![OpenCV](https://img.shields.io/badge/OpenCV-4.5+-blue.svg)

## 功能特性

### 🛠️ 核心功能
1. **图像分块 (Split)**
   - 将大尺寸图像按指定尺寸分割成小块
   - 支持自定义分块宽度和高度（默认 1024x1024）
   - 自动验证图像尺寸是否能被分块尺寸整除
   - 批量处理整个文件夹中的图像

2. **图像整合 (Merge)**
   - 将分割的小块图像重新合并为原始大图
   - 智能识别分块文件的命名规则
   - 支持批量处理多个分块序列
   - 保持原始图像质量

3. **质量评估 (Evaluate)**
   - 批量比较预测图像与真实图像（GT）的质量
   - 计算 PSNR (峰值信噪比) 和 SSIM (结构相似性) 指标
   - 生成详细的 CSV 评估报告
   - 支持批量处理图像对

4. **单图对比 (Single Evaluation)**
   - 可视化对比处理图与真值图
   - 实时滑动条调整对比分割位置
   - 实时显示 PSNR 和 SSIM 评估结果
   - 直观的图像差异对比界面

   - 点击"开始评估"按钮

#### 👁️ 单图对比
1. 在"单图对比"标签页中：
   - 点击"浏览..."选择处理图（预测图）

   - 实时查看对比效果和评估指标

## 使用说明

### 图像分块
- **输入**: 支持 PNG, JPG, JPEG, BMP 格式
- **输出**: 每个图像生成多个分块文件，命名格式为 `原文件名_行号_列号.jpg`
- **分块尺寸**: 建议根据实际需求调整，常用 512x512, 1024x1024, 2048x2048

### 图像整合
- **输入**: 包含分块图像的文件夹
- **输出**: 合并后的完整图像
- **自动识别**: 程序会自动识别分块文件的命名规则并正确排序

### 质量评估
- **评估指标**:
  - **PSNR (峰值信噪比)**: 数值越高表示质量越好，单位 dB
  - **SSIM (结构相似性)**: 数值越接近 1 表示结构相似性越高
- **输出报告**: CSV 文件包含每对图像的评估结果

### 单图对比
- **实时对比**: 选择图片后立即显示对比效果
- **滑动调整**: 拖动滑块调整左右对比区域
- **直观评估**: 红色分割线显示对比边界

## 常见问题

### Q: 支持哪些图像格式？
A: 支持 PNG, JPG, JPEG, BMP 格式。建议使用无损格式如 PNG 以获得最佳质量。

### Q: 分块尺寸应该如何选择？
A: 根据您的具体需求选择：
- **小尺寸 (256-512)**: 适合网络传输或小规模处理
- **中等尺寸 (1024-2048)**: 平衡质量和处理速度的推荐选择
- **大尺寸 (4096+)**: 适合高分辨率图像处理

### Q: 为什么我的图像无法分块？
A: 请确保图像尺寸能够被分块尺寸整除。例如，如果分块尺寸是 1024x1024，则图像的高度和宽度都应该是 1024 的倍数。

### Q: 如何解读 PSNR 和 SSIM 值？
A: 
- **PSNR**: 数值越高越好，通常 30dB 以上表示质量很好，20-30dB 一般，低于 20dB 质量较差
- **SSIM**: 数值越接近 1 越好，通常 0.9 以上表示结构相似性很高
