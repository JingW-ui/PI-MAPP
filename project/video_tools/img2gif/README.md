# 图片转GIF工具

这是一个简单的图形界面应用程序，可以将多张图片转换为 GIF 动画。

## 功能特点

- 支持多种图片格式：PNG, JPG, JPEG, BMP, TIFF
- 可视化界面操作
- 可调节 GIF 每帧持续时间
- 实时预览选中的图片
- 支持添加、删除和清空图片列表

## 使用方法

1. 运行程序：
   ```
   python img2gif.py
   ```

2. 点击"添加图片"按钮选择要转换的图片文件
3. （可选）在图片列表中选择图片并点击"移除选中图片"来删除不需要的图片
4. 在"每帧持续时间"中设置每张图片的显示时间（毫秒）
5. 点击"转换为GIF"按钮开始转换
6. 选择保存位置和文件名完成转换

## 界面说明

- **添加图片**：选择一个或多个图片文件添加到列表中
- **移除选中图片**：删除列表中选中的图片
- **清空所有图片**：清除整个图片列表
- **转换为GIF**：开始将图片列表转换为 GIF 文件
- **预览区域**：显示当前选中图片的缩略图
- **每帧持续时间**：设置 GIF 中每张图片的显示时间（毫秒）

## 依赖库

- PyQt5：用于创建图形界面
- Pillow：用于图像处理

安装依赖：
```
pip install pyqt5 pillow
```

## 注意事项

- 转换过程可能需要一些时间，取决于图片数量和大小
- 程序会在转换过程中显示进度条
- 如果遇到错误，会弹出错误提示框
